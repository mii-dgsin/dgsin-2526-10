Renewable electricity integration

This integration combines the local CO₂ emission records stored in MongoDB Atlas with external yearly electricity generation data provided by Ember Energy.

The frontend calls the project backend, and the backend acts as a proxy to the external Ember Energy API. This avoids exposing the external API key in Angular.

  • Local source: carbon-emission-records
  • External source: Ember Energy API
  • Proxy route: /api/v1/integrations/renewable-electricity
  • Location selector: dynamic compatible locations from both sources
