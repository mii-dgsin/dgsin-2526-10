Analytics

This page lists the analytical views and widgets implemented in the project. The charts are generated with Highcharts and use both the local CO₂ emissions API and the external Ember Energy integration.

CO₂ emissions by year

This chart shows the evolution of total CO₂ emissions by year using the local carbon-emission-records API stored in MongoDB Atlas.

  • Source: local API
  • Widget: Highcharts visualization
  • Metric: total CO₂ emissions
  • Default example: Spain, 2020-2023
Open CO₂ emissions chart

Ember electricity data by year

This chart shows yearly electricity generation data obtained from the external Ember Energy API through the project backend proxy.

  • Source: Ember Energy API
  • Widget: Highcharts visualization
  • Metric: yearly electricity generation
  • Default example: Spain, 2020-2023
Open Ember electricity chart

CO₂ emissions vs electricity generation

This combined chart relates the local CO₂ emissions dataset with the external Ember Energy dataset in a single visualization using two Y axes.

  • Source: local API + external API
  • Widget: Highcharts combined chart
  • Purpose: compare both datasets in one view
  • Default example: Spain, 2020-2023
