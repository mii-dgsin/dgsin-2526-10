Implemented functionality

  • REST API deployed on Google App Engine.
  • Persistence with MongoDB Atlas.
  • Angular frontend with routing.
  • CRUD operations for CO₂ emission records.
  • External integration with Ember Energy through a backend proxy.
  • Highcharts visualizations.
  • Postman tests and public Postman documentation portal.
